## üß≠ The 5 Dimensions Defined

1.  **Product**: How the end-user interacts with the system (CLI, Reports, React Web UIs).
2.  **Data**: How state, context, and long-term memory are managed (CSVs, FAISS Vectors, Databases).
3.  **Logic**: The deterministic code, routing, and orchestration rules connecting components (Genkit flows, Pydantic).
4.  **AI**: The probabilistic engines, guardrails, and models doing the reasoning (Gemini, Llama, Nomic).
5.  **Infrastructure**: Where and how the code is built, tested, and hosted (Docker, Codespaces, Cloud Run).
