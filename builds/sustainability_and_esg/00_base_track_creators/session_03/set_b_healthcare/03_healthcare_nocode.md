## Introduction

Welcome to the **Base Track** for Sustainability & ESG, a comprehensive no-code solution designed to help organizations effectively manage their social and environmental responsibilities.

In this workshop, you will learn how to construct an n8n workflow with three parallel or sequential AI Agent nodes: Analyst, Auditor, and Reporter. This setup enables you to monitor and analyze sustainability metrics, verify compliance, and generate actionable insights into your ESG performance.

## Step-by-Step Guide

1. **The Trigger**: Add a Webhook Trigger node (to accept queries) that receives input from external data sources.
2. **The Analyst Node**: An AI Agent node prompted to analyze Sustainability & ESG trends using machine learning models trained on relevant datasets.
3. **The Auditor Node**: An AI Agent node prompted to verify compliance and detect potential issues by analyzing transactional data.
4. **The Reporter Node**: An AI Agent node that synthesizes the output into a markdown report highlighting key findings, recommendations, and insights.

*An n8n template `workflow_session_03.json` is provided in this directory.*
